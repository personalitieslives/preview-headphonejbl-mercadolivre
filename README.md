🎧 JBL Quantum 100M2 — Página de Review (Landing Page)

Este projeto é uma landing page moderna e responsiva desenvolvida em HTML e CSS para apresentar uma análise completa do headset JBL Quantum 100M2, com foco em conversão (vendas) através de um link afiliado.

📌 Sobre o Projeto

A página foi criada com o objetivo de:

Apresentar um review detalhado do produto
Destacar benefícios, especificações e diferenciais
Exibir provas sociais (reviews de usuários)
Conduzir o usuário até a ação de compra (CTA)
Servir como uma página de afiliado otimizada para conversão
🚀 Funcionalidades
🎯 Estrutura de landing page focada em vendas
📱 Design 100% responsivo
⚡ Animações suaves (CSS)
🧠 Copywriting estratégico (gatilhos mentais)
⭐ Sistema visual de avaliação (estrelas e notas)
📊 Barras de desempenho (qualidade, conforto, etc.)
🧾 Seção de prós e contras
💬 Depoimentos de usuários
🔗 Botões de CTA (Call To Action) com link afiliado
📌 Botão flutuante (sticky CTA)
🛠️ Tecnologias Utilizadas
HTML5
CSS3 (puro, sem frameworks)
Design responsivo com flexbox e grid
Animações com @keyframes
